Review - Mentor Based Programming Training

2016-11-15
programming, reviews

Source: Three Sixty GH

Different people have different ways to pick up something new. As such, there is no ‘one-size’ fits all solution.

When learning to program, there are many ways. One way is video based courses. I wrote a blog post earlier on them.

In my experience, the most effective way to learn to program though is via a mentor. Someone who could plan your learning path and guide you throughout the way.

Here’s a list of mentor based programming resources that I have come across in the past few years.

1. Thinkful

Courses: Web development bootcamp, Angular, NodeJS, Python, Rails and iOS using Swift

Individual courses are around $300 per month. Bootcamp is $9000 for 6 months

2. Bloc

Courses: Software developer track(full and part time), Rails, Android,UX/UI and iOS

Software Developer Track (Full-Time): $19,500.

Software Developer Track (Part-Time): $8,800.

Other courses: Around $5,000.

3. The Firehose Project

Courses: Rails based software development curriculum

Accelerated Pace(24 weeks): $6500.

Moderated Pace(44 weeks): $8500.

4. Code Union

Courses: Fundamentals of Web Development, Rails, TDD, & Professional Engineering and Topics in Computer Science

Each individual web development workshop lasts 8 weeks and costs $3,000.

5. Coding Dojo

Courses: PYTHON,LAMP,MEAN, iOS & RUBY/RAILS

Online bootcamp: $6000

6. Launch School (Tealeaf Academy)

Courses: Quite different from other resources mentioned here. Launch Academy offers a path to become a competent software developer. The path has about 5 courses which cover HTML, JS, Ruby/Rails , Algorithms, etc.

Their course takes somewhere around 700 - 1100 hours to complete. They charge $199 per month.

7. Others:

http://www.practicalcoding.in/
https://careerfoundry.com
http://codifyacademy.com/
https://breakoutmentors.com/
http://www.codecloud.me/
https://learn.moderndeveloper.com/

Best kept secrets of programmers

2016-04-23
point by point, programming

This is my collection of quotes on how to be a better programmer.

  1. Know the concepts. Solving a problem via memory or pattern recognition is much faster than solving it by reason alone.

  2. Know the tools. This is not an end in itself, but a way to maintain “flow” while programming. People who learn their tools, practice using their tools, and automate things that the tools can’t do by themselves can easily be several times as productive as those who do none of those things.

  3. Manage time. If you want to write code, write code. If you want to review a bunch of patches, review a bunch of patches. Don’t try to do all three together. Get your mind set to do one thing, then do that thing for a good block of time before you switch to doing something else.

  4. Reuse everything. Reuse ideas. Reuse code.

Point by Point - Cordova and Ionic

2016-04-14
Ionic, point by point, programming

Cordova

  1. Apache Cordova enables programmers to build applications for mobile devices using CSS3, HTML5, and JavaScript

  2. Just straight HTML,CSS and Javascript, no platform-specific ‘native’ APIs like those in Android, iOS, or Windows Phone

  3. Cordova applications execute within wrappers targeted to each platform, and rely on standards-compliant API bindings to access each device’s capabilities such as sensors, data, network status, etc.

  4. The resulting applications are hybrid, meaning that they are neither truly native mobile application (because all layout rendering is done via Web views instead of the platform’s native UI framework) nor purely Web-based (because they are not just Web apps, but are packaged as apps for distribution and have access to native device APIs).

Understand the basics of NodeJs The Feynman Way

2016-03-05
NodeJS, programming

Understand the basics of NodeJS using the Feynman Technique(Whats that? Read here).

1. Concept

A simple google search for NodeJS would return:

  • NodeJS is a software platform which can run programs written in javascript.
  • Its an open-source, cross-platform runtime environment for developing server-side Web applications.
  • The JavaScript is executed by the V8, a JavaScript engine developed by Google which is used in Chrome browser.
  • It uses a JavaScript API to access the network and file system.
  • NodeJS uses asynchronous programming and has an evented I/O model.
  • It is singlethreaded and lock free. This is what makes it really fast.

The Feynman Technique to learn better

2016-03-04

The Feynman Technique helps you to understand, recall and explain anything in under 20 minutes.

It works on the principle that learning is not about remembering something difficult, but it is about making things easier.

The Feynman technique essentially has five steps: Concept, Explain Like I’m 5, Pinpoint, Analogy and Simplify.
